レクトパズル

84th nameko

84期namekoです。自作ゲームレクトパズルを紹介します。

※注意点
　実はこのゲームを作るときに使っていたパソコンが壊れたり、
夏休み中に様々な誘惑に 翻弄されたりした結果、制作期間が2週間になってしまいました。
つまり、文化祭の２週間前である9/14から作り始めました…。
なので、この部誌の内容と当日展示するゲーム内容が若干変わっている可能性があります。
「まだまだ時間はあるから余裕」 と思っていた昔の自分を叱りたい…

どんなゲーム？

〇マス×〇マスの方眼四角形のマス目を一定のマスずつ数回で塗っていって、
すべてのマスを塗りつぶすゲームです。
ただし、一回のうちで隣り合ったマスは塗れません。 というルールです。 もうちょっと具体的に説明すると、
「５×５の25マスの方眼を3マスずつ塗っていく」
というルールを選んだとします。

このマスから３つのマスを選びます。 ただし、選んだマスが他のマスの周囲に入っていてはいけません。
↓これはOK（×は選べないマス）

↓これはアウト

一度選んだマスはもう選べません。
この作業を繰り返して9回で全てのマスを埋めます（目標）。

残ったマスが全て選べないマスの場合もゲーム終了です。

どのようにして作ったの？

processingというプログラミング言語を使いました。
この言語は図形を動かすのに適しています。
今年、うちの部活のプログラミング展示品の大半がこの言語を使っています。

なぜこんなゲームに？

前回の文化祭では、複数人でプレイできるゲームを展示したのですが、
操作が複雑で、高いパソコン操作能力が必要なゲームでした。
そこで、今回は「操作が簡単」、「初心者でもできる」、
そして「面白くて中毒性がある」ゲームを目指して作りました。
制作期間が２週間だと感じさせないような凄いゲームに、期待してください！
