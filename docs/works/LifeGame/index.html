ライフゲームとは何か

ライフゲームとは、画面に並んでいるマス(セル)の生き死にをシミュレーションするものです。

ルール

一つのセルの周りの8つのセルのうち、生きているセルの数を数えます。
その数が0または1ならそのセルは死に、2ならその状態を維持、3なら生きている状態になり、4以上だと死にます。
この動作を全てのセルで行います。

作成するために使ったもの

processingというプログラミング言語を使いました。

感想

本来、スーパーマリオもどきのような感じのものを作る予定だったのですが、時間的に難しいと思いこれにしました。
画面の上端と下端の処理を省略してしまったところが心残りなので、次作るときは、省略せずに作りたいです。
